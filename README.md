# Deploy to GitLab

## Navigation
| Keybinding | Action |
| --- | --- |
| Arrow Left | Collapse Parent |
| Arrow Right | Expand Node |
| Arrow Down | Highlight Down |
| Arrow Up | Highlight Up |

# Controls
| Keybinding | Action |
| --- | --- |
| Space | Toggle Completion |
| Tab | Increase Indent |
| Shift + Tab | Decrease Indent |
| Alt + Up | Move Up |
| Alt + Down | Move Down |
| Return | Edit/End Edit |
| Escape | Cancel Edit |
| Ctrl + Q | Quit |
