# Deploy to GitLab

## Global
| Keybinding | Action |
| --- | --- |
| Ctrl + Q | Quit |

## Navigation
| Keybinding | Action |
| --- | --- |
| Left | Collapse Parent |
| Right | Expand Node |
| Down | Highlight Down |
| Up | Highlight Up |
| Alt + Left | Collapse Parent (Recursive) |
| Alt + Right | Expand Node (Recursive) |
| Alt + Down | Move Down |
| Alt + Up | Move Up |

## Editing
| Keybinding | Action |
| --- | --- |
| Space | Toggle Completion |
| Return | Edit/End Edit |
| Escape | Cancel Edit |
| Tab | Increase Indent |
| Shift + Tab | Decrease Indent |
